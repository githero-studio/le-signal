# Le Signal

Un développeur reçoit un commit qu'il n'a jamais écrit. Le SHA est impossible. Le message ne contient qu'une coordonnée GPS. À 3h du matin, seul dans un open space désert, il va devoir choisir : ignorer, enquêter, fuir ou alerter.

**Auteur** : Git Hero Studio
**Langue** : fr
**Classification** : 16+
**Tags** : thriller, dev, mystère, cyberpunk

> ⚠️ Ce livre contient des thèmes anxiogènes : surveillance, disparition, paranoïa numérique.
